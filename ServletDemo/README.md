# ServletDemo Project
- this is web project using Servlet :
- create Dynamic web project
- create AddServlet which will have ONLY doGet() method which handle incoming request  
- Ask user to enter 2 numbers , when press Sum this /add request will go to AddServlet to handl it by doGet() method
- fetch parameters from the request , then apply sum to get result
- Response will breturn back to the client with the result 

  ### Prerequisites
  - java 25
  - Servlet-api 6.1.0
  - STS Eclipse
